# Customer Retention & Churn Analysis Dashboard using Excel

## 📊 Project Overview
This project focuses on analyzing customer churn and retention patterns using a telecom dataset. The goal is to identify why customers leave and provide actionable insights to improve retention.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Visualization (Charts & Dashboard)

## 📁 Dataset
- Telco Customer Churn Dataset

## 🔍 Key Insights
- Customers with month-to-month contracts have the highest churn rate
- New customers (0–12 months tenure) are more likely to churn
- Payment methods influence customer retention behavior

## 💡 Recommendations
- Offer incentives for long-term contracts
- Improve onboarding for new users
- Encourage automatic payment methods

## 📌 Dashboard Features
- KPI Cards (Total Customers, Churn Rate, etc.)
- Churn Analysis by Contract, Tenure, and Payment Method
- Interactive Slicers for filtering data

## 🚀 Outcome
This project demonstrates skills in data cleaning, analysis, and dashboard creation, simulating real-world business scenarios.
